# Bipolar Brain

雙相情緒障礙症（Bipolar Disorder）每日文獻日報，由 AI 自動彙整 PubMed 最新論文。

## 架構

- **PubMed E-utilities API** → 抓取最新雙相情緒障礙症相關文獻
- **Zhipu GLM-5.1** → 繁體中文摘要、PICO 分析、臨床實用性評估
- **GitHub Actions** → 每日台北時間 11:00 自動執行
- **GitHub Pages** → 自動部署 HTML 報告

## 涵蓋領域

- 核心精神醫學 / 情緒障礙
- 心理治療 / 心理社會介入
- 神經科學 / 生物標記 / 神經影像
- 物理治療 / 運動 / 復健
- 職能治療 / 功能復原
- 中醫 / 針灸 / 中西整合醫學
